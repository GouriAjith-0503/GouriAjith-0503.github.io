EcoMatrix – Hardware-Accelerated Environmental Monitor

Built an environmental monitor using a Nexys 4 FPGA and ESP32, achieving sub-millisecond response times for safety alerts via parallel Verilog logic. Integrated a "Region-Aware" feature allowing adaptation to global climate zones (Tropical/Arid) via hardware toggles. Deployed a full-stack cloud solution using Cloudflare Workers and React to track UN Sustainable Development Goal metrics (Clean Water & Air).
